О курсе Web разработка на Python/Django PyBursa

Django - среда с открытым исходным кодом для ведения разработки Web-приложений в стиле модель-вид-контроллер (model-view-controller - MVC), использующая язык Python.
С помощью Django вы можете в считанные минуты создавать высококачественные, легкоуправляемые Web-приложения, работающие под управлением баз данных.

Программа курса

Блок 1
Основы Python как высокоуровневого языка программирования общего назначения.
Поддержка Python нескольких парадигм программирования.

Блок 2
Процесс создания веб-сайта или веб-приложения. Основные этапами процесса. Веб-дизайн, вёрстка страниц, программирование для веб на стороне клиента и сервера, а также конфигурирование веб-сервера.

Блок 3
Cвободный фреймворк для веб-приложений на языке Python Django.
Использование шаблона проектирования MVC.
Сайт на Django из одного или нескольких, отчуждаемых и подключаемых приложений.
